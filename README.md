# Data-Structures-and-Algorithms
Increasing expertise in data structures through learning and practicing problems. 

## Ramblings

Most people sigh AHHHHH when DSA is mentioned but DSA is actually really cool when you learn it as methods to solve problems. It could be a subsitute to a daily suduko/crossword. 

Agreed, if you give me a leetcode hard/medium which I have not seen before and ask me to solve it in 15 mins I will fumble, but would it be something fun to solve without the added must should would? Yes. 

As a developer in an MnC, did DSA help improve my work? 
YES! Even though in many cases I do not directly use DSA it does help improve code comprehension and quality. It is like grease to an engine. 

## Structure

I have divided this repo into multiple folders for each broad data structure and added a short readme with my understanding from solving problems. Practice exercises contains problems from leetcode/neetcode which I find to be a good base for understanding.
